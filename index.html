Git init : Untuk menginisialisasi repository dan memberitahu folder yang udijadikan repository

Git config : untuk konfigurasi email dan username

Git status : untuk mengetahui status file

Git add . : untuk membuat file ke stagging area

Git commit : untuk menambahkan pesan pada file yang telah dirubah

Git log : untuk melihat riwayat commit, untuk keluar tekan (q)

Git clone : untuk mengcopy file yang ada di github

Git push : untuk mengirimkan commit ke server

Git pull : untuk mengambil commit dari server

Git remote : untuk berinteraksi dari client ke server atau sebaliknya

Git branch : untuk membuat branch baru

Git branch --all : untuk melihat semua branch

Git chekout nama_branch : untuk berpindah branch

Git branch -d nama_branch : untuk menghapus branch
